Процедура прохождения Верификации

Для прохождения верификации пользователь высылает свое фото с ID документом в руке на этой странице в поле «прикрепить документ». Примерно так, как ниже

Файл .jpg .jpeg .png

Информация на документе должна быть читаемой, чтоб можно было провести сверку.

При проверке будут сверяются:


- фотография



Имя и Фамилия при прохождении проверки будут приведены в соответствие с вашим ID документом.


Администратор или кто-то другой не может скопировать присланный вами документ, а после проверки независимо от результата – присланный вами документ автоматически удаляется из системы без возможности его последующего восстановления. Мы понимаем ответственность данного шага.


Документа для идентификации принят системой и будет рассмотрен в ближайшие 24ч. О результатах идентификации вам будет сообщено дминистрацией Проекта.
