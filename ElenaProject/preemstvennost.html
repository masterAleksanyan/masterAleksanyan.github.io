Суть преемственности

  • Владелец аккаунта имеет возможность передать право пользования своим аккаунтом любому третьему лицу, по своему усмотрению. Для этого он может оформить своё волеизъявление, заполнив прилагаемую форму. Заверение этой формы происходит: либо непосредственно на нашем Ресурсе, цифровой подписью (персональным кодом) владельца, либо письменно оформляется у адвоката или нотариуса и сканированный формат заверенного документа в электронной форме прикрепляется вложением в прилагаемую форму.
  • Владелец аккаунта имеет возможность передать право пользования своим аккаунтом непосредственно администрации Ресурса, по наступлению какого–либо конкретного события или по истечению конкретно указанного времени для последующего помещения этого ЛК и/или его части в ЦБ. Для этого он может оформить своё волеизъявление, заполнив прилагаемую форму. Заверение этой формы происходит: либо непосредственно на нашем Ресурсе, цифровой подписью (персональным кодом) владельца ЛК, либо письменно оформляется у адвоката или нотариуса и сканированный формат заверенного документа в электронной форме прикрепляется вложением в прилагаемую форму.
  • Владелец аккаунта имеет возможность самостоятельно открыть свой Личный Кабинет пользователя, переслав его весь целиком или только определённую его часть, в ЦБ, не передавая никому никаких прав на пользование им, для нахождения ЛК на Ресурсе в ЦБ без ограничения по времени и доступу. Для этого он может оформить своё волеизъявление, заполнив прилагаемую форму.
